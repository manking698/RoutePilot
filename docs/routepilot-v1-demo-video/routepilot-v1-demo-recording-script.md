# RoutePilot V1 Demo Script

Target length: 45-60 seconds.

1. Open RoutePilot.
2. Show current location on the map.
3. Tap the search / destination input.
4. Select Shah Alam I-City as the demo destination.
5. Show the route preview.
6. Show ETA and distance.
7. Tap Go.
8. Show the navigation instruction card.
9. Tap the navigation instruction card to show Route steps.
10. Scroll the route steps and show the final arrival card.
11. Tap the navigation instruction card again, or tap Hide, to close Route steps.
12. Move / simulate location.
13. Show marker following the route.
14. Show arrived state.
15. End demo.

Recording notes:

- Use the default demo route from Taman Kinrara to Shah Alam I-City.
- Keep the route preview, navigation card, speed, and arrival state visible.
- Keep Route steps hidden by default, then briefly show it during navigation.
- Avoid showing private API keys, terminal windows, or unrelated device notifications.
